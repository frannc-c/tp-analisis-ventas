# Analisis de ventas

## Integrantes
- Hugo
- Paco
- Luis

## Objetivo
Analizar un conjunto de datos de ventas para obtener indicadores básicos y representar la evolución de las ventas en el tiempo.

## Estructura
- datos/
- scripts/
- resultados/

## Ejecución
1. Abrir el proyecto en Google Colab.
2. Ejecutar `scripts/analisis_ventas.py`.
3. Ejecutar `scripts/grafico_evolucion_ventas.py`.
4. Revisar los archivos generados en `resultados/`.

## Dataset
Se utilizó el archivo `ventas.csv`, ubicado en la carpeta `datos/`.

## Resultados generados
- `resumen_ventas.txt`
- `evolucion_ventas.png`
